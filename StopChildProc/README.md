## StopChildProc

Программа ищет и замораживает дочерние процессы.

![Окно программы](.images/image2.png)

1. Имя искомого процесса (например firefox.exe, Code.exe);
2. Задержка между циклами поиска и заморозки дочерних процессов (в мс);
3. Задержка после первого обнаружения искомого процесса (в мс);
4. Окно вывода дерева замороженных процессов.

![Пример использования программы](.images/image3.png)
